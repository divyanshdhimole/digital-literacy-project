## 📋 Module Summaries

### ✅ Task 1 — Digital Literacy Awareness Infographic `(Module 1 | 20 marks)`
Created a one-page infographic using **Canva** covering three topics:
what digital literacy is, safe internet practices, and professional online
presence. The design uses a clean two-column layout with icons and short
bullet points to make key information accessible to first-year students.

📁 Files in: `task-1-presentation/`

### ✅ Task 2 — Student Digital Portfolio `(Module 2 | 20 marks)`
Set up foundational professional profiles on three platforms:

- **GitHub** — Profile README with name, branch, year, and learning goals
- **LinkedIn** — Education section filled with current degree and institution
- **Stack Overflow** — Account created to engage with the developer Q&A community

📁 Files in: `task-2-portfolio/`
